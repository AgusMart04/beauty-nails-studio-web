# Beauty & Nails Studio

Sitio web para el salón de uñas Beauty & Nails en Alicante.

## Características

- Manicura y pedicura profesional
- Uñas acrílicas y de gel
- Citas online
- Galería de trabajos
- Información de contacto

## Tecnologías

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui
